# Assets Folder

Expo requires these image files:
- icon.png (1024x1024)
- splash.png (1284x2778)
- adaptive-icon.png (1024x1024, Android only)
- favicon.png (48x48, web only)

For now, Expo will use default placeholders. We'll add custom icons later.

To add custom icons:
1. Create your images at the sizes above
2. Save them in this folder
3. Restart the dev server
